🌸 Support Vector Machine (SVM) Classification on Iris Dataset

This project demonstrates the implementation of a Support Vector Machine (SVM) model using the famous Iris dataset in Python. The assignment focuses on data exploration, visualization, model training, prediction, and performance evaluation using machine learning techniques.

📌 Project Overview

In this notebook, the Iris dataset was analyzed and classified using the SVM algorithm provided by Scikit-learn. Different visualization techniques were used to better understand the dataset before training the model.

The project also includes hyperparameter tuning using GridSearchCV to find the best model parameters and improve model performance.

📊 Tasks Performed

Evaluating the model using:
Confusion Matrix
Classification Report
Applying GridSearchCV for parameter tuning
📈 Results

The SVM model achieved very high classification accuracy on the Iris dataset. GridSearchCV was used to test different values of C and gamma, confirming the optimal parameters for the classifier.
